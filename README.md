# STAT596 South Bay Pollution Project

This repository contains the notebooks, cleaned datasets, figures, and tables for my STAT 596 final project:

**Water, Air, and Weather Drivers of Sewage-Related Pollution Exposure in South San Diego County: A Spatiotemporal Exploratory Analysis**

## Project Overview

This project examines sewage-related water-quality and air-quality indicators in South San Diego County using public monitoring datasets. The analysis combines beach advisory and closure records, Tijuana River flow, rainfall, H2S monitoring records, RTMA weather variables, beach monitoring station metadata, census tract boundaries, and a 1 km monitoring-priority grid.

The project is exploratory. It does not claim causation or prediction. The goal is to identify spatial and temporal patterns that may support environmental monitoring, public communication, and future planning.

## Repository Structure

- `notebooks/` - Final notebooks used for data cleaning, analysis, statistics, and mapping.
- `data/cleaned/` - Cleaned datasets used in the analysis.
- `data/final/` - Final analysis-ready datasets.
- `data/spatial/` - Spatial files used for maps and the 1 km grid.
- `figures/` - Final figures and maps used in the report.
- `tables/` - Report-ready summary tables and statistical outputs.
- `appendix/` - Supporting material for the final paper.
